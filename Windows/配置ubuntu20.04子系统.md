#### 安装gedit

为了方便修改配置文件，喜欢用gedit的就可以安装。

```bash
sudo apt-get install gedit
```

#### 换国内源

1.备份原来的source文件

```bash
sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak 
```

2.修改sources.list文件

```bash
sudo gedit /etc/apt/sources.list
```

3.添加国内源，选择其一就可以了。
