# ubuntu安装最新版gcc

### 环境

ubuntu20.04

### 安装步骤

#### 1.更新包列表

```bash
sudo apt update
```

#### 2.安装build-essential安装包

```bash
sudo apt install build-essential
```

#### 3.打印gcc版本验证是否安装成功

```bash
gcc --version
```
